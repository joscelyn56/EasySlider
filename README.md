# Easyslider
create a slideshow with no CSS or JS, just style your container and add classes to begin
```html
  <div class="easy-slider easy-hide-dots">
			<div class="easy-slider-body">
				  <div class="easy-slide active">
              //slide design
				  </div>
          ...
				  <div class="easy-slide">
              //slide design
				  </div>
			</div>
	</div>
```
